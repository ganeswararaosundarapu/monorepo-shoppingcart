# monorepo-shoppingcart
monorepo for shoppingcart

# How to run the monorepo
## Install
 yarn install-all

## Build
 yarn build-all

## Run UI
 yarn start-ui

## Scripts
 Run: yarn [ScriptName]

 echo-all - echo's from all the packages if echo exist in the package script
 build-all - builds for all the packages
